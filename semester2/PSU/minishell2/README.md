# Minishell 2

Minishell 2

shell working with basic command like ls.

built-in cd:
"cd .." = father folder
"cd -" = last open folder
"cd ." = current folder
"cd ~" = home/usr
"cd" = home/usr
"cd path" = ~/path
"cd incorrect-path" = path: No such file or directory.
"cd path1 path2" = cd: Too many arguments.
